receive :: Nat -> a -> ReceiverHalf a -> (ReceiverHalf a, [a]) Source #

Process one arriving frame and return the updated state and any newly deliverable payloads.

  • Duplicate (n < expected): dropped silently, no state change.
  • Out-of-order (n > expected): buffered in pending, no output.
  • In-order (n == expected): delivered, then consecutive entries flushed from pending.
